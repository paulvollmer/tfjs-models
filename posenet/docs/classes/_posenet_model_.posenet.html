estimateMultiplePoses

  • estimateMultiplePoses(input: InputType, imageScaleFactor?: number, flipHorizontal?: boolean, outputStride?: OutputStride, maxDetections?: number, scoreThreshold?: number, nmsRadius?: number): Promise<Pose[]>
    • Defined in posenet_model.ts:218

    Infer through PoseNet, and estimates multiple poses using the outputs. This does standard ImageNet pre-processing before inferring through the model. The image should pixels should have values [0-255]. It detects multiple poses and finds their parts from part scores and displacement vectors using a fast greedy decoding algorithm. It returns up to maxDetections object instance detections in decreasing root score order.

    Parameters

    • input: InputType

      ImageData|HTMLImageElement|HTMLCanvasElement|HTMLVideoElement) The input image to feed through the network.

    • Default value imageScaleFactor: number = 0.5

      A number between 0.2 and 1. Defaults to 0.50. What to scale the image by before feeding it through the network. Set this number lower to scale down the image and increase the speed when feeding through the network at the cost of accuracy.

    • Default value flipHorizontal: boolean = false

      Defaults to false. If the poses should be flipped/mirrored horizontally. This should be set to true for videos where the video is by default flipped horizontally (i.e. a webcam), and you want the poses to be returned in the proper orientation.

    • Default value outputStride: OutputStride = 16

      the desired stride for the outputs. Must be 32, 16, or 8. Defaults to 16. The output width and height will be will be (inputSize - 1)/outputStride + 1

    • Default value maxDetections: number = 5

      Maximum number of returned instance detections per image. Defaults to 5.

    • Default value scoreThreshold: number = 0.5

      Only return instance detections that have root part score greater or equal to this value. Defaults to 0.5

    • Default value nmsRadius: number = 20

      Non-maximum suppression part distance in pixels. It needs to be strictly positive. Two parts suppress each other if they are less than nmsRadius pixels away. Defaults to 20.

    Returns Promise<Pose[]>

    An array of poses and their scores, each containing keypoints and the corresponding keypoint scores. The positions of the keypoints are in the same scale as the original image
