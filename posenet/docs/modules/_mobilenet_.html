toOutputStridedLayers

  • toOutputStridedLayers(convolutionDefinition: ConvolutionDefinition[], outputStride: OutputStride): Layer[]
    • Defined in mobilenet.ts:131

    Takes a mobilenet architectures' convolution definitions and converts them into definitions for convolutional layers that will generate outputs with the desired output stride. It does this by reducing the input stride in certain layers and applying atrous convolution in subsequent layers. Raises an error if the output stride is not possible with the architecture.

    Parameters

    • convolutionDefinition: ConvolutionDefinition[]
    • outputStride: OutputStride
